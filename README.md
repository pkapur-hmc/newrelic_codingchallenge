### Pratyush Kapur Coding Challenge

## Steps to run the application
* `./gradlew build`
* `./gradlew shadowJar`
* `./ java -jar ./build/libs/coding-challenge-shadow.jar` - the first time this is run it creates the server, every time after that it creates a client.
